# Solaris Energia - Versão CNPJ 33.456.490/0001-94

Esta pasta contém uma versão do site institucional da **Solaris Energia** voltada ao público corporativo, vinculada ao CNPJ **33.456.490/0001-94** e com textos adaptados para empresas, comércios e indústrias.

## Visão Geral

O foco desta variação é apresentar a energia solar como estratégia de redução de custos e fortalecimento de práticas ESG em negócios de diferentes portes.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, com a mesma identidade visual sustentável da versão base.
-   **Uso típico:** Landing page para campanhas B2B, apresentações comerciais e anúncios segmentados.

## Funcionalidades

As mesmas funcionalidades da pasta `solaris-energia-static` são mantidas aqui:

1.  Header com mudança visual ao rolar;
2.  Menu mobile responsivo;
3.  Scroll suave entre seções;
4.  Simulação de envio do formulário;
5.  Animações de entrada nos cards de soluções.
